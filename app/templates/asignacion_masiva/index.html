¿Cómo Funciona?

Esta herramienta automatiza varias tareas en un solo paso:

  • Validación de Empleados: Verifica que cada pasaporte exista en la base de datos.
  • Creación Automática: Si una Cartera o Grupo mencionado en el archivo no existe, el sistema lo creará automáticamente.
  • Asignación Inteligente: Asigna cada empleado al grupo especificado. Si ya está asignado, lo ignora.
  • Horarios Especiales: Si se proporcionan fechas y horas, se crearán o actualizarán los registros de horarios especiales para el grupo correspondiente.
  • Manejo de Errores: El sistema procesará todas las filas válidas e informará cuántas tuvieron errores (ej. pasaporte no encontrado, formato de fecha incorrecto).
